# Assignment 5 - Web

This assignment consisted of two independent web projects focused on 
backend web architecture, session management, and database integration
using Java Servlets, JSP, Apache Tomcat, and MySQL.

## User Login System

A lightweight authentication web application simulating login and registration workflows.

* User login and account creation flaws
* Validation for incorrect credentials and duplicate usernames
* Dynamic welcome pages with personalized content
* Central AccountManager class for credential storage and verification

## Basic Store Website

A dynamic e-commerce style web application backend by a MySQL product database

* Homepage dynamically listing products from the database
* Individual product pages using URL parameters
* Per-user **session-based shopping cart**
* Add to cart functionality with quantity updates and removal
* Real time total price calculation
* Product image rendering via database file references
